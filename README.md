# 网页下载扩展 Page Download 

> 支持 Manifest V3, 浏览器扩展插件, 用于下载网页资源(html, css, js, img, ...)

> 之前用的[梦想网页资源下载器](https://github.com/ryanker/dream_downloads)是基于被淘汰的[Manifest V2](https://blog.chromium.org/2024/05/manifest-v2-phase-out-begins.html)开发导致现在在最新的浏览器中已无法使用，这里基于Manifest V3创建新的扩展插件。
